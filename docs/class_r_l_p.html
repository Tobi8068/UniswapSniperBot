Detailed Description

template<std::size_t BufferSize = 9, std::size_t ElementsCount = 1024>
class RLP< BufferSize, ElementsCount >

Disables validation for faster processing speed Class for Recursive Length Prefix Encoding. It is the main encoding method used to serialize objects in Ethereum.

Template Parameters

ElementsCountNumber of elements to encode. Ethereum transactions require 9: nonce, gas price, gas limit, to, value, data, v, r, s
